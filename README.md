## Run Locally

1. Clone the repository:
```bash
  git clone <link>
```
2. Navigate to the project directory: 
```bash 
cd fuclothes 
```
3. Install dependencies: 
```bash 
npm install
```
4. Start the development server: 
```bash
npm run dev
```
